## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-check-accessibility-enabled`
- `allow-open-accessibility-settings`
- `allow-get-frontmost-ui-tree`
- `allow-click-node`
